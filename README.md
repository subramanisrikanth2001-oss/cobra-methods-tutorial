# COBRA Methods Tutorial in COBRApy

This repository reproduces classical COBRA/FBA tutorial examples using COBRApy.

## Contents

- Toy metabolic network reconstruction
- Growth on alternate substrates
- Production of cofactors and biomass precursors
- Alternate optimal solutions
- Robustness analysis
- Phenotypic phase plane analysis
- Simulating gene knockouts
- Network flexibility analysis

## Files

### notebooks/Toy_model.ipynb
Manual reconstruction of the toy metabolic network.

### notebooks/Supplementary_examples_1_8.ipynb
Implementation of supplementary COBRA method examples.

## Software

- Python
- COBRApy
- NumPy
- Pandas
- Matplotlib
